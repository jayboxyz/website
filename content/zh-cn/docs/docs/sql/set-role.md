设置角色
========

摘要
--------

"```{.none}"
SET ROLE（角色|全部|无）
```

问题描述
-----------

`SET ROLE`设置当前目录中当前会话的启用角色。

`SET ROLE role`使当前会话具有单个指定的角色。若要使`SET ROLE role`语句成功，执行该语句的用户应具有给定角色的权限。

`SET ROLE ALL`使当前用户在当前会话中被授予的所有角色生效。

`SET ROLE NONE`将禁用当前会话中授予当前用户的所有角色。

限制
-----------

部分Connector不支持角色管理。有关更多详细信息，请参阅连接器文档。
