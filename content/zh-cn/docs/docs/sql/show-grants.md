展览奖赏
===========

摘要
--------

"```{.none}"
展示授权【表名】表名
```

问题描述
-----------

列出当前用户对当前目录中指定表的授权。

如果未指定表名，则该命令将列出当前用户对当前目录的所有架构中的所有表的授权。

该命令要求设置当前编录。


**说明**

*执行任何授权命令前，请确保已启用认证*

示例
--------

在表`orders`上列出当前用户的授权：

餐桌上展示的赠品；

列出当前用户对当前目录下所有模式中所有表的授权：

展览赠品；

限制
-----------

有些连接器不支持`SHOW GRANTS`。有关更多详细信息，请参阅连接器文档。
