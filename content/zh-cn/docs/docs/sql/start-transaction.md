裁武会谈
=================

摘要
--------

"```{.none}"
开始事务【模式[,...】]
```

其中`mode`是

"```{.none}"
隔离级别{读未提交|读已提交|可修复读|串行化}
读取{仅|写入}
```

问题描述
-----------

为当前会话启动一个新的事务。

示例
--------

``` {.sql}
裁武会谈；
开始事务隔离级别可重复读
开始事务读写；
开始事务隔离级别读已提交，只读；
开始交易读取写，隔离水平序列化；
```

参见
--------

[commit](./commit)，【回退】（./回退），提交
