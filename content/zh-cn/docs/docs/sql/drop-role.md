落杆
=========

摘要
--------

"```{.none}"
DROP ROLE角色名称
```

问题描述
-----------

`DROP ROLE`删除当前目录中的指定角色。

`DROP ROLE`语句要成功，执行它的用户必须拥有给定角色的管理员权限。

示例
--------

删除角色`admin` ：

删除角色管理；

限制
-----------

部分Connector不支持角色管理。有关更多详细信息，请参阅连接器文档。

参见
--------

【创建角色】（./create-role），【设置角色】（./set-role），【授权角色】（./grant-roles），【回收角色】(./revoke-roles)
