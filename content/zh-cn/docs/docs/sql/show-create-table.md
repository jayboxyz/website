显示创建表
=================

摘要
--------

"```{.none}"
SHOW创建TABLE的表名
```

问题描述
-----------

显示创建指定表的SQL语句。

示例
--------

显示可以用来创建`orders`表的SQL：

创建表sf1.orders；

"```{.none}"
创建表
-----------------------------------------
创建表tpch.sf1.orders(
订单键大
订单状态varchar，
总价翻倍，
订单日期变量
)
有(
format = 'ORC'，
partitioned_by = ARRAY【'订单日期'】
)
（1行）
```
