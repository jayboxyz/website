描述输出
===============

摘要
--------

"```{.none}"
DESCRIBE输出语句_名称
```

问题描述
-----------

列出预备语句的输出列，包括列名（或别名）、目录、模式、表、类型、类型大小（以字节为单位）和是否别名列的布尔值。

--------

从国家选择

DESCRIBE输出结果为my_select1 ；
```
