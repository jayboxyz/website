查询资源
==============

Query REST服务是其余服务中最复杂的。它
包含节点的详细信息，以及其他详细信息。
捕获在Presto上执行的查询的状态和历史
安装。

- GET /v1/查询

此服务返回有关查询的信息和统计信息，这些查询是
目前在Presto协调器上执行。

当你将浏览器指向一个Presto坐标时，你会看到一个渲染的
此服务输出的版本，将显示最近的
在Presto安装上执行的查询。
   

-获取/v1/查询/{queryId}信息

如果您希望收集有关查询的非常详细的统计数据，
这就是你要调用的服务。加载的Web界面
presto协调器，您将看到当前查询的列表。单击
查询将揭示到此服务的链接。
